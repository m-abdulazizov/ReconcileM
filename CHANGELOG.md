# Changelog

All notable changes to ReconcileM are documented here.

## 0.1.0-SNAPSHOT

### Added

- Multi-module Gradle project structure.
- Core reconciliation model and engine.
- Rule-based matching with scores and explanations.
- Built-in matching rules for amount, currency, date, reference, and counterparty checks.
- Record normalization support.
- CSV reader and CSV result writer.
- JDBC record reader.
- JDBC result persistence repository.
- Spring Boot starter with auto-configuration.
- Example application with CSV and JDBC demo endpoints.
- `reconcilem-test` fixtures and assertions.
- Project documentation for architecture, matching rules, CSV, JDBC, Spring Boot, and extension points.
- GitHub Actions CI workflow.

### Planned Before First Stable Release

- Maven publishing finalization.
- Stronger integration and error-path tests.
- More specific exception hierarchy.
- Release notes for a tagged version.
